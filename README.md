# Steady Command Center

This project is a Vite + React + TypeScript application.

## Tooling

- Runtime: `node`
- Canonical package manager: `npm`

The repository is standardized on the `packageManager` field in [package.json](/Users/andrea/Documents/coding/steady-command-center/package.json).

## Getting Started

1. Install Node.js 20 or newer.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm test
```

## Notes

- `npm` is the team default for this repo.
- Legacy `bun.lock` files may still exist from earlier work, but `npm` is the supported workflow going forward.
